# 💫 About Me:

I am a Computer Engineering student specializing in the intersection of Full-Stack Web Development, Immersive 3D/Motion Graphics, and Cloud Deployment. 

Core Expertise
Web Development: Experienced in architecting robust systems using the MERN and PERN stacks, integrating secure payment processors like Stripe, and deploying applications on AWS and cloud platforms.
Creative Coding & Animation: Crafting immersive visual environments and fluid, timeline-driven layouts using Three.js, GSAP, Anime.js, and Particle.js to bring interfaces to life.
Machine Learning & Systems: Designing dependable data-driven workflows, optimizing features using TensorFlow, and developing cross-platform engineering applications.
Programming: Proficient in JavaScript, TypeScript, Java, Python, C++, HTML, and CSS.

Projects & Impact
Tomato: A full-stack MERN food application featuring secure Stripe payment gateway integration, engineered for scalable performance and deployed on AWS.
Prescripto: A specialized full-stack platform built to streamline and modernize medical management and appointment scheduling workflows.
gsap_cocktails: An interactive, motion-heavy front-end showcase leveraging advanced GSAP animations and fluid UI transitions.
ChemEquip: A hybrid web and desktop analytics suite designed for interactive equipment parameter visualization.
